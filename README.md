## Event Driven Microservice Case Study

Event Driven Microservice Beginner Case Study using Spring Boot & RabbitMQ

### Technology Stack

- Java 11
- Spring Boot
- Spring Data JPA
- RabbitMQ
- Lombok
- H2 Database
- Docker (Compose)

### Running the Application

```sh
  $ docker compose up
```
